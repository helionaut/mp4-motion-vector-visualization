Browser MVP

Inspect the split pair and judge the motion story directly.

This first slice keeps the workflow on the client machine. It plays a user-provided MP4 pair in the browser and overlays an honest optical-flow approximation computed from decoded frames in a worker.

1. Source selection

The UI can accept a single Insta360 source file, but this MVP does not yet perform a truthful browser-side split into the derived MP4 pair.

2. Pair upload fallback

Upload the already-derived pair to continue the browser demo without server-side processing.

3. Playback and overlay

Visualization mode: optical-flow-approximation
